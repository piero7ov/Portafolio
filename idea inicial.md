Vamos a crear un portafolio web con HTML, CSS y JavaScript. Que se alimente de un json con la informacion de los proyectos.

Tendremos un archivo index.html, un archivo style.css y un archivo script.js.

El json tendra la siguiente informacion:
proyectos
- titulo
- fecha_de_creacion
- descripcion
- tecnologias
- imagen
- enlace_al_repositorio


El estilo sera responsive, y a la antigua. Adaptaremos estilos de la interfaz de Windows 95. Ademas, el inicio sera, una pantalla de una consola para poder acceder al portafolio. Habra que hacer un "pip install pierodev" para acceder al portafolio. Y para que la gente se pueda guiar habra un "pip install pierodev" para que la gente se pueda guiar, habra un "no module installed pierodev", de alguna manera guiarlos para que instalen el pierodev y que tenga un poco de gracia el portafolio.
